###### 六级标题

字体

*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___

分割线

在一行中用三个以上的星号、减号、底线来建立一个分隔线。

下划线

可以通过HTML的​<u>​标签来实现

<u>带下划线的内容</u>

首行缩进空格

（1） 一个空格大小的表示：&ensp;或者&#8194;
（2） 两个空格大小的表示：&emsp;或者&#8195;

列表

MarkDown 支持有序列表 和 无序列表

无序列表

无序列表使用星号(*****)、加号(+)或是减号(-)作为列表标记

有序列表

有序列表使用数字并加上 . 号空格来表示

列表嵌套

列表嵌套只需在子列表中的选项添加四个空格即可

1.parent1
    +p1-child1
    +p1-child2
2.parent2
    -p2-child1
    -p2-child2

区块

在段落开头使用 > 符号 ，然后后面紧跟一个空格符号
区块是可以嵌套的

> MarkDown

>> MarkDown

区块于列表结合

1.区块中使用列表
2.列表中使用区块

> 区块中使用列表
> 1. 有序
> + 无序
> 2. 有序
> - 无序

1. 列表中使用区块
    > 区块内容
    >> 区块内容
2. 列表中使用区块

链接

简单使用

[链接名称](链接地址)
或者
<链接地址>

// 例：
[百度](www.baidu.com)
<www.baidu.com>

高级链接

链接也可以用变量来代替，文档末尾附带变量地址

百度[百度][1]
教育宝[教育宝][jyb]

[1]:www.baidu.con
[jyb]:https://www.jiaoyubao.cn/
